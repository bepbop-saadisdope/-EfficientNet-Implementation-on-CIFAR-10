# EfficientNet Implementation on CIFAR-10

## Objective
Implement EfficientNet-B0 from the paper *"EfficientNet: Rethinking Model Scaling for Convolutional Neural Networks"* (Tan & Le, ICML 2019) and evaluate its performance on CIFAR-10 compared to baseline architectures (ResNet-18 and a simple 3-layer CNN).

## References
- **Paper:** [EfficientNet: Rethinking Model Scaling for CNNs (arXiv:1905.11946)](https://arxiv.org/abs/1905.11946)
- **Dataset:** [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html)
